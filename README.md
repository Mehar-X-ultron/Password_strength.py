# Password Strength Checker (Simple)

A beginner-friendly Python tool to evaluate password strength.  
Estimates entropy, checks for common patterns, and provides actionable suggestions.

> **Educational only** — Do not use this as a replacement for professional password management solutions.

## Features
- Entropy estimate (heuristic)
- Checks for common passwords and weak patterns
- Interactive and one-shot CLI modes
- Clear, human-readable suggestions

## Requirements
- Python 3.7+

## Usage
Interactive mode:
```bash
python3 password_checker.py
